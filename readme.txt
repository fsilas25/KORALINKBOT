KoraLink AI Chatbot
===================

Project Name:
-------------
KoraLink AI Assistant

Description:
------------
This chatbot helps users learn about KoraLink, the KoraLink Marketplace,
Digital Community Champions, products, training, commissions, and digital services.

Technology Stack:
-----------------
- Python
- Flasks
- HTML
- CSS
- JavaScript

Features:
---------
✓ Website chatbot interface
✓ KoraLink FAQ system
✓ Agent information
✓ Marketplace information
✓ Commission information
✓ Product information
✓ Training information
✓ Inventory information
✓ Sales tracking information

Installation:
-------------

1. Create virtual environment:

   python -m venv venv

2. Activate environment:

   Windows:
   venv\Scripts\activate

3. Install requirements:

   pip install -r requirements.txt

4. Run app.py:
    it will process all data

5. Open browser:
    http://127.0.0.1:5000

-------------------------
You will see bot icon at right-bottom corner click and ask your question related to KoraLink
